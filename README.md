## 🔅 ‘HAVIT’이란? 🔅
습관을 의미하는 영단어 Habit과 가지다를 의미하는 Have it을 합쳐 “습관을 들이다”라는 의미를 함축한 프로젝트명입니다. 계획한 목표를 공유해서 같이 습관화할 크루원을 모집할 수 있고, 목표를 달성하지 못한 멤버를 독려해 동기부여를 도와주는 목표 공유 모바일 웹 서비스입니다.
